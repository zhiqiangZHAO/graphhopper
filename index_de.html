GraphHopper: ein Routenplaner in Java

GraphHopper stellt effiziente Algorithmen für die Routenplanung bereit. Z.B. Dijkstra oder A*, aber auch optimierte Algorithmen wie Contraction Hierarchies. GraphHopper steht under der Apache Lizenz und die Bibliothek wird dauerhaft, automatisch auf Fehler überprüft, dennoch ist sie im Alpha-Status, d.h. GraphHopper selbst ist stabil, die API ist es noch nicht.
Die Basis
GraphHopper kann Server-seitig, Desktop-seitig oder für Android eingesetzt werden.
  • Klein und schnell
  • Speichereffizient
  • Sehr gut getestet
Fürs Web
GraphHopper-Web ist ein schlanker Webservice der auch ein kleines Abfrageinterface bereitstellt.
